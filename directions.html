آموزش استفاده از Git برای دانشجویان درس GPS

در این آموزش، یاد می‌گیرید چطور ریپوی زیر را کلون کرده و همیشه به‌روز نگه دارید:

https://github.com/atoofihub/GPS_UT_TA

۱. نصب Git

برای ویندوز:

  1. به سایت https://git-scm.com/download/win بروید.
  2. نصب‌کننده (Installer) را دانلود و نصب کنید (تنظیمات پیش‌فرض را تأیید کنید).
  3. پس از نصب، برنامه Git Bash را اجرا کنید.

برای مک (macOS):

  1. ترمینال (Terminal) را باز کنید.
  2. دستور زیر را وارد کنید:
    3. xcode-select --install
  3. این دستور ابزارهای توسعه را نصب می‌کند که شامل Git نیز هست.

برای اوبونتو (Ubuntu):

  1. ترمینال را باز کنید.
  2. دستور زیر را وارد کرده و رمز عبور خود را وارد کنید:
    3. sudo apt update
sudo apt install git

۲. کلون کردن ریپو

در مرحله بعدی، باید ریپو را به سیستم خود منتقل (clone) کنید.

در ترمینال یا Git Bash دستور زیر را اجرا کنید:

git clone https://github.com/atoofihub/GPS_UT_TA.git

با این کار پوشه‌ای به نام GPS_UT_TA در مسیر فعلی ایجاد می‌شود.

۳. رفتن به داخل پوشه

cd GPS_UT_TA

۴. دریافت آخرین تغییرات (Pull)

هر بار که استاد یا همکلاسی شما تغییری در ریپو ایجاد کردند، می‌توانید آخرین نسخه را با دستور زیر دریافت کنید:

git pull

۵. نکات تکمیلی

  • برای اجرای دستورات بالا نیاز به اتصال اینترنت دارید.
  • اگر بار اول است که Git را روی سیستم اجرا می‌کنید، بهتر است نام و ایمیل خود را تنظیم کنید:
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

۶. سوالات متداول

❓ من ریپو را کلون کرده‌ام. برای به‌روزرسانی چه کنم؟
فقط وارد پوشه شوید و دستور git pull را بزنید.

❓ من پوشه‌ای ندارم. چه کنم؟
ابتدا با git clone ریپو را دریافت کنید.

❓ Git نصب نیست.
به بخش نصب Git در این صفحه برگردید و سیستم‌عامل خود را انتخاب کنید.
