Project Descriptions

Project 1 (P1): Satellite Orbit Visualization and Runge-Kutta Method

Students implemented orbital mechanics calculations using the Runge-Kutta method to visualize and determine satellite orbits. This project focused on fundamental orbital mechanics principles and numerical integration techniques.

Project 2 (P2): Navigation File Reading and Satellite Coordinate Interpolation

Students developed algorithms to read satellite navigation files and perform coordinate interpolation for 30-second intervals between epochs. This project emphasized data processing and interpolation techniques in satellite positioning.

Project 3 (P3): Ionospheric Correction using Klobuchar Model

Students implemented ionospheric correction algorithms using the Klobuchar model to improve GPS positioning accuracy. This project focused on atmospheric correction models and their impact on positioning precision.

Project 4 (P4): Tropospheric Correction using Collins Model

Students applied tropospheric correction using the Collins model to enhance GPS positioning accuracy. This project emphasized atmospheric modeling and correction techniques for improved positioning results.

Project 5 (P5): RINEX Observation File Reading and C1C Code Extraction

Students developed algorithms to read RINEX observation files and extract C1C codes from GPS satellites. This project focused on data format understanding and signal processing in satellite navigation.

Project 6 (P6): Absolute Positioning using SPP Method

Students implemented Single Point Positioning (SPP) algorithms using the previous four projects to determine absolute positions. This comprehensive project integrated all previous techniques for complete positioning solutions.
