Keplerian Elements Description

Main Orbital Parameters

√a (Square root of semi-major axis): This parameter represents the size of the orbit. Its value is equal to the square root of the semi-major axis of the orbital ellipse.

e (Eccentricity): This parameter determines the shape of the orbit and shows the deviation from a perfect circle.

i₀ (Orbital inclination): The angle between the orbital plane and the Earth's equatorial plane.

Ω (Right ascension of ascending node): The angle that determines the position of the intersection point of the orbit with the equatorial plane.

ω (Argument of perigee): The angle between the ascending node and the perigee point in the orbital plane.

M₀ (Mean anomaly): Specifies the angular position of the satellite at the reference time.

Correction Parameters

Cuc, Cus: Latitude argument correction coefficients

Crc, Crs: Orbital radius correction coefficients

Cic, Cis: Inclination correction coefficients

Time Parameters

toe: Ephemeris reference time (seconds in week)

a₀: Satellite clock offset (seconds)

a₁: Satellite clock drift rate (seconds/second)

a₂: Satellite clock drift rate acceleration (seconds/second²)

Additional Parameters

i̇: Rate of inclination angle

L2: P-code on L2 frequency

GPS Week: GPS week number

TGD: Group delay

IODC: Clock data issue number

transmission_time: Message transmission time
