RINEX Observation File: Structure and Example

1. What is a RINEX Observation File?

RINEX (Receiver Independent Exchange Format) is a standard text format for storing GNSS (Global Navigation Satellite System) data, including GPS, GLONASS, Galileo, and BeiDou. The observation file (extension .rnx or .obs) contains raw measurements from a GNSS receiver, organized in a way that is independent of receiver brand.

3. Observation Types

Each GNSS system (GPS, GLONASS, Galileo, BeiDou) has its own set of observation types. These are listed in the header and define the columns in the observation data. For example:

SystemObservation Types
GPS (G)C1C, L1C, S1C, C1W, L1W, S1W, C2X, L2X, S2X, C2W, L2W, S2W, C5X, L5X, S5X, C1X, L1X, S1X
GLONASS (R)C1C, L1C, S1C, C1P, L1P, S1P, C2C, L2C, S2C, C2P, L2P, S2P, C3X, L3X, S3X
Galileo (E)C1X, L1X, S1X, C8X, L8X, S8X, C6X, L6X, S6X, C7X, L7X, S7X, C5X, L5X, S5X
BeiDou (C)C2I, L2I, S2I, C8X, L8X, S8X, C6I, L6I, S6I, C7Z, C7I, L7Z, L7I, S7Z, S7I, C5X, L5X, S5X, C1X, L1X, S1X

Legend: C = Code, L = Carrier phase, S = Signal strength, numbers = frequency bands, X/P/C = signal type.

4. Epochs and Observation Data

Each epoch represents a measurement time. The epoch line starts with a ">" and gives the date, time, and number of satellites observed. Each following line lists the observations for one satellite, with values in the order defined by the header. Example:

> 2024 01 01 00 00  0.0000000  0 34        .000000015974
G01  20888756.039   109771298.977 8        48.750    20888756.369   109771289.968 7        43.000 ...
G02  23201822.669   121926668.150 7        43.250    23201822.179   121926528.154 5        34.750 ...
...
  • Epoch line: > 2024 01 01 00 00 0.0000000 0 34 means year 2024, Jan 1, 00:00:00, 34 satellites observed.
  • Satellite line: Each line starts with the satellite PRN (e.g., G01), followed by the observation values for that epoch.
